Windows 8

  1. Windows 8 - операційна система американської корпорації Microsoft сімейства Microsoft Windows , представлена ​​в 2012 році. Підтримку припинено у січні 2016 року. Windows 8 стала першою операційною системою сімейства, яка була повністю адаптована під сенсорні екрани із самого початку. Для цього Microsoft розробила спеціальний графічний інтерфейс (що нагадує інтерфейс Windows Phone 7 ), що надає доступ до базових функцій. Окремої версії системи для планшетів не передбачалося – система стала універсальною та встановлювалася на всі доступні типи пристроїв. Разом з виходом Windows 8 запрацював магазин програм Windows Store , створений за прообразом аналогічного магазину Apple для користувачів Mac, він дозволив купувати софт, не відкриваючись від екрана комп'ютера.
    Після релізу Windows 8 Microsoft оголосила про перехід на більш швидкий період розробки нових версій своєї ОС - наступна Windows, з кодовою назвою Blue, повинна була вийти через рік після Windows 8. Так воно і сталося - реліз цієї системи побачив світ у жовтні 2013 року, практично рівно через рік після появи Windows 8. Правда, в ході розробки Microsoft вирішила відмовитися від кодового імені та привласнила релізу порядковий номер 8.1.
  2. Існує чотири редакції Windows 8.1 – втім, для звичайного користувача актуальними будуть лише дві з них. Windows RT 8.1 поставляється у вигляді попередньо встановленої ОС на ARM-пристроях (і не дозволяє встановлювати десктопні програми), а Windows 8.1 Enterprise призначена для корпоративних користувачів та поширюється за відповідними ліцензіями. Таким чином, залишається дві редакції: звичайна Windows 8.1 і функціональніша Windows 8.1 Pro. Pro-версія підтримує більший обсяг оперативної пам'яті (512 ГБ проти 128 ГБ), можливість завантаження з віртуального образу VHD, а також пропонує розширену функціональність у плані управління (можливість підключення до доменів, підтримка групових політик тощо) – втім, У разі домашнього використання ця функціональність явно буде надмірною.
  3. Windows 8.1 може встановлюватися як «з нуля», так і як оновлення до Windows 8. У першому випадку користувачеві доведеться розщедритися на $120 (за стандартну версію) або $200 (за Professional). Якщо ж у вас вже є Windows 8, оновлення дістанеться безкоштовно – його можна встановити з фірмового магазину Windows Store в сучасному інтерфейсі (у десктопному Центрі оновлення Windows воно недоступне; мабуть, щоб користувачі не прийняли його за звичайний сервіс-пак). У процесі оновлення зберігаються файли користувача, налаштування, інформація про облікові записи та десктопне ПЗ (додатки для сучасного інтерфейсу, придбані в Windows Store, доведеться встановлювати заново). Сама процедура виконується порівняно швидко та у фоновому режимі, що не заважає використанню ПК. Системні вимоги у Windows 8.1 здебільшого залишилися тими самими, що й у Windows 8, тому практично будь-який комп'ютер із встановленою Windows 8 може бути оновлений до 8.1, за одним невеликим, але суттєвим винятком – 64-бітна версія Windows 8.1 вимагає підтримки CMPXCHG16b, яка у ранніх 64-бітових процесорів AMD (Athlon 64 X2, Opteron 185) була відсутня. Крім того, з такою ж проблемою можуть зіткнутися і власники Intel Core 2 - хоч самі процесори і підтримують ці інструкції, але деякі старі плати - немає. В результаті при спробі оновлення таких систем користувачі одержують повідомлення про помилку "CPU does not support CompareExchange 128". Виходом із цієї ситуації буде використання 32-бітної версії Windows 8.1.
  4. Про це «нововведення» не написав, мабуть, тільки лінивий, але все ж таки повторимося: так, у Windows 8.1 кнопка «Пуск» повернулася на своє законне місце. Схоже, таки незадоволених її відсутністю виявилося досить багато для того, щоб Microsoft зважилася піти на такий крок і, по суті, визнала помилковість свого первісного рішення. Втім, це насправді лише півзаходи – повернулася тільки кнопка «Пуск», але не старий стиль меню, оскільки ця кнопка відкриває той самий головний «плитковий» екран. І так само, як у Windows 8 її не можна було повернути на місце (штатними засобами, принаймні, оскільки альтернативних додатків, що копіюють функціональність стартового меню з попередніх Windows, було випущено достатньо), так і в Windows 8.1 не можна цю кнопку відключити .
  5. Можливості налаштування «плиткового» стартового меню в Windows 8 були досить скромними, тому логічно, що в Windows 8.1 розробник надав більше можливостей конфігурації цього важливого елемента нової ОС під себе. Так, у Windows 8.1 з'явилися нові розміри плиток - "маленькі" і "великі", причому у другому випадку в такій плитці може бути більше інформації (якщо розробник це передбачив). Для персоналізації екрану доступно більше колірних варіантів, плюс при бажанні можна використовувати ті ж шпалери, що і в десктопному оточенні (у такому разі фон залишатиметься статичним при перегортанні «плиток»). Кнопка "вниз" тепер переводить користувача в екран програм, в якому можна вибрати групу програм і закріпити її на початковому екрані. Плитки тепер можна поєднувати в групи, яким можна давати довільні назви – правда, у пошуку вони не використовуються, тому призначені, схоже, виключно для полегшення візуального сприйняття.
  6. Якщо користувач працює за кількома комп'ютерами, йому, напевно, знадобиться можливість синхронізації початкових екранів на всіх системах, де він використовує свій обліковий запис Microsoft.
  7. Як відомо, всі програми в сучасному інтерфейсі запускаються виключно в повноекранному режимі. У Windows 8 за бажання можна було розмістити поряд два вікна, проте при активній роботі цього часто все одно недостатньо. У Windows 8.1 ця функціональність була доопрацьована, з'явилися нові співвідношення розмірів двовіконного інтерфейсу (наприклад, обидва вікна можуть ділити екран між собою порівну).
    Крім того, на великих екранах з'явилася можливість розміщувати поряд не дві програми, а три і навіть чотири - знову-таки, з вибором різних співвідношень розмірів для кожної програми.
  8. Панель управління в сучасному інтерфейсі в Windows 8 була, м'яко кажучи, неактуальною - в ній були представлені лише базові налаштування системи, і тому майже завжди виявлялося, що потрібно звертатися до традиційної десктопної панелі управління, якщо потрібно зробити щось хоч трохи складніше заміни шпалери робочого столу. У Windows 8.1 Microsoft постаралася виправити ситуацію, зробивши інструмент "Параметри комп'ютера" більш функціональним.
    Так, головний екран цієї програми тепер містить список налаштувань, до яких ви часто зверталися в минулому, в «Комп'ютері та пристрої» зосереджені налаштування, що були раніше в меню «Персоналізація», до яких додалися додаткові параметри пристроїв, на кшталт швидкості скролювання миші. Меню «Користувачі» тепер перейменовано в «Облікові записи» і набуло більш впорядкованого вигляду, керування пошуком здійснюється з власного розділу, також тут є мережеві налаштування, керування дозволами для програм, мовні параметри та інше. - Загалом, сучасні «Параметри комп'ютера» стали набагато функціональнішими, ніж раніше. Втім, з десктопною панеллю управління ця програма все одно тягатися не може - але тепер звичайний користувач стане до неї звертатися набагато рідше, ніж у Windows 8.
  9. З Windows 8.1 поставляється нова версія браузера від Microsoft - Internet Explorer 11. У порівнянні з IE10 він став ще швидше (детальніше питання продуктивності буде розглянуто трохи нижче), а також очікувано обзавівся новими можливостями. "Сучасна" версія тепер дозволяє тримати одночасно відкритими до 100 вкладок (раніше їх число обмежувалося абсурдними десятьма), плюс користувач може запустити ще один сеанс сучасного IE11, в якому, у свою чергу, відкрити ще сотню вкладок. Браузер тепер краще відпрацьовує на сенсорних пристроях – зокрема, зі спливаючими вікнами на сайтах і полями введення, пропонує розширену функціональність у спливаючій панелі і т.д.
    Десктопна частина Internet Explorer 11 зовні не змінилася, всі нововведення ховаються «під капотом». Так, ще в Developer Preview, випущеному в червні 2013 р., з'явилася підтримка WebGL та протоколу SPDY, покращене масштабування на екранах з високим DPI, були представлені оновлені інструменти розробників, покращена підтримка HTML5 тощо. Підвищення швидкості роботи та «чуйності» браузера, яке відзначають багато користувачів, пов'язане з реалізацією передзавантаження сторінок та визначення ресурсів, які мають бути завантажені у фоновому режимі для прискорення серфінгу.
    Release Preview, що вийшов наприкінці вересня, представив подальші поліпшення продуктивності, і судячи з номера версії, реліз IE11, що відбувся разом з виходом Windows 8.1, нічим від Release Preview не відрізняється.
  10. Власний хмарний сервіс зберігання даних SkyDrive стає невід'ємною частиною Windows – тепер він встановлений у системі за промовчанням і використовується для зберігання налаштувань, створюваних документів та знімків із камери.
    Крім того, десктопний клієнт SkyDrive більш неактуальний – сервіс інтегрований у Провідник, що також полегшує його постійне використання, а налаштування SkyDrive доступні у своєму розділі в панелі керування (тут показується доступний обсяг, вибираються варіанти синхронізації файлів та системних налаштувань тощо). .
